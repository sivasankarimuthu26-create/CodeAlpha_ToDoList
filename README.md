# CodeAlpha To-Do List (C++)

## 📌 Overview
Console-based To-Do List application built using C++. It supports adding tasks, viewing pending/completed tasks, marking tasks as completed, and saving data to a file.

## 🚀 Features
- ➕ Add tasks
- 📋 View pending tasks
- ✅ View completed tasks
- ✔️ Mark tasks as completed
- 💾 File handling (tasks saved in tasks.txt)

## 🛠️ Tech
- C++
- OOP (Classes)
- STL (vector)
- File I/O

## ▶️ How to Run
g++ main.cpp -o todo  
.\todo
